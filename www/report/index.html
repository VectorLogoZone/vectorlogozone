Logos

  • Inventory - list of every available logo.
  • Missing Logos - list of logos that do not have both rectangle and icon sizes.
  • Tags: Logos by Tag, Report - Unfortunately, the tags aren't well maintained.
  • Visual Inventory - view all the logos at once (WARNING: a lot of images!).

Metadata

  • Colors - detailed color report.
  • Fonts - detailed font report.
  • Metadata Report - what links are available for each logo.
  • Least social - logos with the no social media links
  • Most social - logos with the most social media links
  • Unknown metadata keys - to help cleanup garbage entries in the metadata
  • Notes - that should have been removed before publishing
  • First letter - Counts by first letter

Comparisons

  • ar21 vs card - Make sure the raster versions (with rsvg) look the same as the vector.
  • icon vs favicon - Compare icon vs favicon
  • icon vs tile - Compare the icons and tiles.
  • tile vs SuperTinyIcons - Compare the tiles with images from edent/SuperTinyIcons.
  • icon vs SVGPorn - Compare the icons with images from SVGPorn.
